
  # Memorial Website for Martyrs

## Our Mission

We believe that memory is sacred. This platform honors those we've lost by creating a living, searchable memorial—a space where their stories endure, their contributions are recognized, and their legacies continue to inspire. We're building a space that transcends borders and languages, ensuring that all voices and all memories are preserved with dignity.

---

## What We're Building

A multilingual, accessible memorial platform where communities can:
- **Preserve stories** — Share and discover meaningful narratives about the people we remember
- **Search and connect** — Find memorials across a unified archive, available in multiple languages
- **Honor legacies** — Celebrate contributions, add tributes, and keep memories alive for future generations

## Tech Stack

Built with modern, sustainable technologies:

- **React 18** — Fast, component-based UI
- **Vite** — Lightning-fast development and builds
- **TailwindCSS** — Beautiful, utility-first styling
- **React Router** — Seamless, client-side navigation
- **React Query** — Reliable data fetching and caching
- **i18next** — Multilingual support (Arabic & English)
- **Framer Motion** — Smooth, meaningful animations
- **Lucide React** — Consistent, beautiful icons
